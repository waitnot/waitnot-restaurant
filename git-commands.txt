# Git Commands to Push WaitNot to GitHub

# 1. Initialize Git repository (if not already done)
git init

# 2. Add all files to staging
git add .

# 3. Create initial commit
git commit -m "Initial commit: WaitNot Restaurant Ordering System

Features:
- Complete restaurant management system
- QR code ordering for customers
- Real-time order tracking with WebSocket
- Dual printer system (Kitchen + Cash Counter)
- Comprehensive analytics dashboard
- Multi-language support
- UPI/Card/Cash payment options
- Mobile-responsive design
- JSON file database (production ready)
- Ready for Render deployment"

# 4. Add GitHub remote
git remote add origin https://github.com/waitnot/Waitnot.git

# 5. Set main branch
git branch -M main

# 6. Push to GitHub
git push -u origin main

# 7. Verify push was successful
git status
